Avalon-node

The Nodejs Client For VMarket

avalon-node目前是基于nodejs的模块，安装包都是基于npm库的，所以在安装nodejs之后在命令行输入以下命令进行安装或者更新

安装

win: npm install -g avalon-node
mac: sudo npm install -g avalon-node

或者

git://github.com/czy88840616/avalon-node.git
npm link

更新

(sudo)npm update -g avalon-node

使用

  1. 在命令行输入vm，默认端口为 80 ，mac用户和linux用户请sudo vm，不想用80端口的可以使用vm -p 8080来修改端口
  2. 打开 127.0.0.1:xxxx访问首页进行配置，或者在配置页面配置自动打开浏览器选项
  3. enjoy

参数

  • -p(–port) [number] 端口号
  • help 打开帮助文档
